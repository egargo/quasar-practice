# Quasar App (quasar-project)

(Quasar + Cypress) Practice


## Setup
```bash
# Install dependencies
yarn

# Start the app in development mode (hot-code reloading, error reporting, etc.)
quasar dev

# Lint the files
yarn lint

# Format the files
yarn format

# Build the app for production
quasar build

# Run Cypress
yarn cypress open
```
